XiaokeTools 游戏素材管理器 - 便携版
======================================

【启动】
双击 XiaokeTools.exe 即可运行，无需安装。

【首次使用】
1. 启动后点击「+ 添加库」
2. 选择素材库根目录（如 D:\Xiaoke\GameAssets）
3. 点击「扫描」，等待索引完成（4万文件约5秒）
4. 即可浏览、勾选、导出素材

【主要功能】
- 浏览：分类树 / 素材包 / 缩略图墙（三栏布局）
- 搜索：文件名搜索 + 类型过滤
- 预览：图片(可缩放) / SVG / GIF / 音频 / 字体 / 文本 / 3D模型(glb/obj/dae)
- 导出：勾选素材 → 导出为文件夹或 zip + 版权清单 + 素材索引
- 去重：检测压缩包与解压目录并存，一键清理（可指定备份位置）
- 类型管理：自助新增/编辑资源类型（支持未来新格式）

【数据存储】
- 索引数据库：%APPDATA%\com.xiaoke.tauri-app\index.db
- 缩略图/转换缓存：同目录下
- 卸载：删除本文件夹 + 上述数据目录即可

【系统要求】
Windows 10/11，WebView2 Runtime（Win10/11 通常已自带）
3D 模型的 .blend 预览需本机安装 Blender（可选）
