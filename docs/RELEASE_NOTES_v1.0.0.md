# 笑客宝库 XiaokeVault v1.0.0

游戏素材资产管理桌面工具。基于 Tauri 2 + Vue 3 构建，轻量（安装包仅 4MB）、原生性能。

## 📦 下载安装

**Windows x64**：`XiaokeVault_1.0.0_x64-setup.exe`（4.0 MB）

- 下载后双击安装，无需额外运行时
- SHA256：`834318f1c7b08d812d0a07f508d77f81957ce9b20c0af818ddae664881e7c90d`

## ✨ 主要功能

### 资产管理
- **目录树浏览**：自动扫描库目录结构，按真实文件夹层级展示
- **自动监听**：文件增删改实时检测，增量更新索引（无需手动重扫）
- **多格式预览**：
  - 🖼️ 图片（PNG/JPG/WEBP 等，含缩略图缓存）
  - 🎞️ GIF 逐帧播放 + 帧编辑器
  - 🔊 音频（MP3/WAV/OGG）
  - 📄 文本（TXT/JSON/XML/代码）
  - 🧊 3D 模型（GLB，基于 Three.js）
  - 🔤 字体（TTF/OTF 预览）

### 搜索与筛选
- **全局实时搜索**：输入即搜（500ms 防抖），跨目录匹配文件名
- 搜索状态可视化：spinner 加载提示、无结果空态、未搜索引导
- 点击搜索结果自动定位到文件所在目录

### 选择与导出
- **按目录勾选**：整目录（含子目录）一键选择
- **按文件勾选**：精确控制单个文件
- **项目导出**：
  - 支持文件夹复制 / zip 打包两种格式
  - 自动生成 `manifest.json`（文件清单）和 `CREDITS`（版权信息）
  - 导出结构镜像库内目录树

### 其他
- **自定义资源类型**：扩展名 → kind 分类映射，可自定义
- **双语界面**：中文 / English 一键切换
- **无边框窗口**：现代化 UI

## 🛠️ 技术栈

| 层 | 技术 |
|---|---|
| 框架 | Tauri 2 |
| 前端 | Vue 3 + Pinia + Tailwind v4 + vue-i18n |
| 后端 | Rust + sqlx (SQLite) |
| 预览 | Three.js / gifuct-js / @tanstack/vue-virtual |

## 📋 系统要求

- Windows 10/11 (x64)
- 无需预装任何运行时（WebView2 随系统自带）

## 🐛 已知限制

- 仅支持 Windows（macOS/Linux 暂未适配）
- 不支持视频预览（计划在后续版本加入）

## 📖 使用方式

1. 安装后启动，点击「+ 添加库」选择游戏素材根目录
2. 点击「扫描」建立索引（首次约 5-10 秒，4 万文件级别）
3. 左侧目录树浏览，点击文件右侧预览
4. 勾选需要的文件/目录，点击「导出」打包
